


## For Setup
In the main tree, run setup.bat. This file installs all the required modules and creates the Files folder. 

Now youll go up to the top of the page where it says ip = '127.0.0.1'. You can change this to what you need along with port which is right above it. Most of the time you shouldnt need to change the port number. 
Next go to the code in main.js. The line that is tempLocation: 'C:/Users/Parker Fink/Desktop/tmp/'. You must change it from my name to whatever your PC has your name as.
Now run START.bat and your server should be fully functional.
